## Overview

**Eleccions Generals** is a database project that models and imports official Spanish general election data (2016). It includes:

- A complete MySQL relational schema covering the full geographic and electoral hierarchy
- A Java application that parses official `.DAT` election files and populates the database
- A ZIP extraction utility to unpack archived `.DAT` files
- Categorized SQL query examples demonstrating aggregation, JOINs, subqueries, and window functions

## Data model at a glance

The schema covers the 2016 Spanish general elections with the following core tables:

| Table | Description |
|---|---|
| `comunitats_autonomes` | Spain's 17 autonomous communities |
| `provincies` | 52 provinces with seat (escó) counts |
| `municipis` | Municipalities, linked to provinces |
| `eleccions` | Election events with date |
| `candidatures` | Political parties/coalitions per election |
| `persones` | Individual candidate personal data |
| `candidats` | Candidates linked to a candidacy and province |
| `eleccions_municipis` | Per-municipality vote totals |
| `vots_candidatures_mun` | Per-candidacy votes by municipality |
| `vots_candidatures_prov` | Per-candidacy votes and seats by province |
| `vots_candidatures_ca` | Per-candidacy votes by autonomous community |
