These 10 queries combine data from multiple tables using `INNER JOIN` and `LEFT JOIN`. They link the `candidats`, `persones`, `provincies`, `municipis`, `candidatures`, and `eleccions` tables to answer questions about candidates, parties, and election results.

## Query 1

Show `candidat_id`, name, concatenated surnames with a space between them, and gender displayed as `"Home"` (M) or `"Dona"` (F).

```sql
SELECT c.candidat_id, p.nom, CONCAT(p.cog1, " ", p.cog2) AS cognoms, IF(p.sexe = 'M', "Home", IF(p.sexe = 'F', "Done", NULL)) AS sexe
	FROM candidats c
INNER JOIN persones p ON p.persona_id = c.persona_id;
```

## Query 2

Show candidates from the province of Barcelona: `candidat_id`, `nom`, `cog1`, and `candidatura_id`.

```sql
SELECT c.candidat_id, p.nom, p.cog1, c.candidatura_id
    FROM candidats c
INNER JOIN persones p ON p.persona_id = c.persona_id
INNER JOIN provincies pro ON pro.provincia_id = c.provincia_id
WHERE pro.nom = "Barcelona";
```

## Query 3

How many candidates are there per candidatura? Show `nom_curt` and the count aliased as `"Quantitat de candidats"`.

```sql
SELECT cs.nom_curt, COUNT(c.candidat_id) AS "Quantitat de candidats"
	FROM candidats c
INNER JOIN candidatures cs ON cs.candidatura_id = c.candidatura_id
GROUP BY cs.candidatura_id;
```

## Query 4

Show the full name (`nom_llarg`) of candidatures whose elections took place between January and June.

```sql
SELECT cs.nom_llarg
	FROM candidatures cs
INNER JOIN eleccions e ON e.eleccio_id = cs.eleccio_id
WHERE e.mes BETWEEN 1 AND 6;
```

## Query 5

Total blank votes (`vots_blanc`) in the municipality of Malgrat de Mar in 2016.

```sql
SELECT SUM(em.vots_blanc)
	FROM eleccions_municipis em
INNER JOIN municipis m ON m.municipi_id = em.municipi_id
INNER JOIN eleccions e ON e.eleccio_id = em.eleccio_id
WHERE m.nom = "Malgrat de Mar" AND e.any = 2016;
```

## Query 6

How many women ran as titular (`'T'`) candidates? Show the count aliased as `"Num_Dones"`.

```sql
SELECT COUNT(candidat_id) AS "Num_Dones"
	FROM candidats c
INNER JOIN persones p ON p.persona_id = c.persona_id
WHERE p.sexe = 'F' AND c.tipus = 'T';
```

## Query 7

Show the maximum number of polling tables (`num_meses`) used per municipality, with the municipality name and the alias `"maxim de meses utilitzades"`.

```sql
SELECT m.nom, MAX(em.num_meses) AS "maxim de meses utilitzades"
	FROM eleccions_municipis em
INNER JOIN municipis m ON m.municipi_id = em.municipi_id
GROUP BY m.municipi_id;
```

## Query 8

Show all provinces and their candidate counts, ordered by province name.

```sql
SELECT p.nom AS Provincia, COUNT(c.candidat_id) AS num_candidats
	FROM candidats c
INNER JOIN provincies p ON p.provincia_id = c.provincia_id
GROUP BY p.nom
ORDER BY p.nom;
```

## Query 9

Show the first name and first surname of candidates and the full name (`nom_llarg`) of their candidature, ordered by name then surname.

```sql
SELECT p.nom, p.cog1 AS cognoms, cs.nom_llarg
	FROM candidats c
INNER JOIN persones p ON p.persona_id = c.persona_id
INNER JOIN candidatures cs ON cs.candidatura_id = c.candidatura_id
ORDER BY p.nom, p.cog1;
```

## Query 10

Show each candidate's name and first surname concatenated with a comma, the candidature short name, and the election name for elections in 2016. Use aliases `"candidat"`, `"candidatura"`, and `"eleccio"`.

```sql
SELECT CONCAT(p.nom, " , ", p.cog1) "candidat", cs.nom_curt "candidatura", e.nom "eleccio"
	FROM candidats c
INNER JOIN persones p ON p.persona_id = c.persona_id
LEFT JOIN candidatures cs ON cs.candidatura_id = c.candidatura_id
LEFT JOIN eleccions e ON e.eleccio_id = cs.eleccio_id
WHERE e.any = 2016;
```
